Azure Commit Copy

Build format string

The format string that will be used to generate a string for each build the commit is associated with. Create a format string using the ES6 Template literals syntax. The variables in scope are:
Variable name Description
build.name The name of the Azure build
build.id The ID of the Azure build
build.url The URL of the Azure build

Format string

The format string that will be used to generate a string for the commit. Create a format string using the ES6 Template literals syntax. The variables in scope are:
Variable name Description
branch.name The name of the branch
branch.url The URL of the branch
pullRequest.id The ID of the Azure pull request
pullRequest.url The URL of Azure pull request
commit.hash The SHA1 hash of the commit
commit.url The URL of the commit
buildsString The string made up of the concatenation of the build strings
