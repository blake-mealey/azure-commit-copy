Build format string

The format string that will be used to generate a string for each build the commit is associated with. Create a format string using the ES6 Template literals syntax. The variables you have access to are:

  • build.name: the name of the Azure build
  • build.id: the ID of the Azure build
  • build.url: the URL of the Azure build

Format string

The format string that will be used to generate a string for the commit. Create a format string using the ES6 Template literals syntax. The variables you have access to are:

  • commit.hash: the SHA1 hash of the commit
  • commit.url: the URl of the commit
  • buildString: a string made up of the concatenation of the build strings
